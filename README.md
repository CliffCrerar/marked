# Marked

Markdown render service.

## Endpoint

- `POST`: render markdown text  
  ```javascript
  {
      text: '<MarkdownText>',   // required
      title: '<DocumentTitle>',
      linkCSS: '<URL>',
      inlineCSS: '<CSSText>'
  }
  ```
